本文为个人的学习笔记，整理成专栏，方便自己查漏补缺。

在本栏目中，我将会总结学过的课程，并且着重为城市空间数据研究的初学者提供简洁易懂的Python入门教程。正式阅读本教材之前，请参照配置好阅读可以notebook的环境，并且对城市空间数据研究等概念有一定了解。
